Initialize Repository
git init
Creates a new git repository in current folder
Clone Repository
git clone <url>
git clone https://github.com/user/repo.git
Check Status
git status
Shows changed or staged files
Add Files to Staging
git add .
Adds all modified files
Commit Changes
git commit -m "message"
git commit -m "fixed bug"
View Commit History
git log
Lists previous commits
Create Branch
git branch feature
Creates branch named "feature"
Switch Branch
git checkout feature
Switches to branch "feature"
Create + Switch Branch
git checkout -b dev
Creates and switches to "dev"
Merge Branch
git merge feature
Merges branch feature into current branch
Push to Remote
git push origin main
Uploads commits to remote repository
Pull Latest Changes
git pull
Gets updates from remote
Stash changes
git stash
Temporarily saves changes without committing
Apply Stash
git stash pop
Restores last stashed changes
Undo Last Commit
git reset --soft HEAD~1
Uncommit but keep changes staged
Remove untracked files
git clean -f
Deletes files not in git
